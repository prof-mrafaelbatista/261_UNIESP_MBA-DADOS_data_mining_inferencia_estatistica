## About the Course

This course covers the **practical core of Data Science**: the process of transforming raw, incomplete, and heterogeneous data into reliable and actionable information. Following the **CRISP-DM** cycle, students go through all the steps that precede modeling — steps that, in real-world projects, account for up to **80% of the time** spent on a data project.

The approach is predominantly practical (**70% hands-on**), using **Python on Google Colab**, real-world datasets from multiple domains, and interactive tools (Wayground and Miro) for engagement and collaboration. At the end, each group delivers a **complete preprocessing pipeline** applied to a real problem.

---

## Learning Objectives

| # | Objective |
|---|-----------|
| 1 | Describe the **Data Science Lifecycle** and the professional roles involved (Data Engineer, Data Scientist, and Data Analyst) |
| 2 | **Collect data** from files (CSV, JSON, Excel), REST APIs, and Web Scraping using Python |
| 3 | Perform **Exploratory Data Analysis (EDA)** with Pandas, Matplotlib, and Seaborn, interpreting distributions, descriptive statistics, and variable relationships |
| 4 | **Detect and handle** missing values and outliers with technical criteria and documented justification |
| 5 | Apply **feature engineering** techniques: categorical variable encoding, distribution transformations, and numerical variable rescaling |
| 6 | Understand and apply **statistical inference** concepts: population, sample, sample size, PCA, and hypothesis testing |
| 7 | Deliver a **complete preprocessing pipeline** that is documented, reproducible, and oriented toward a real business problem |
